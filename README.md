# Car Browser

A simple React application for browsing rental cars with search, filters, sorting, and URL synchronization.

## Features

* Search cars by name
* Custom 300ms debounce
* Filter by transmission
* Filter by car type
* Available only toggle
* Sort by price (Low → High / High → Low)
* Live results counter
* Empty state with Reset Filters button
* URL query synchronization
* Responsive layout

## Technologies

* React
* Vite
* React Router DOM
* CSS

## Installation

```bash
npm install
```

## Run the project

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
